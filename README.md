# Layout Builder — Wireframe Prototype

A drag-and-drop, blueprint-style prototype builder for designing screens, pages, and layouts.

## Features

- Drag wireframe components onto a canvas to build a layout
- Resize components from width, height, or corner
- Reorder, remove, or reset size on any placed component
- Add your own components by **describing** them in plain words
- Add your own components by **sketching** them freehand on a drawing canvas
- Export the layout as JSON

## Run locally

Just open `index.html` in any modern browser. No build step.

## Stack

- Plain HTML / CSS / vanilla JS (no framework)
- Figtree font from Google Fonts
- Blueprint visual language: blue line work on lavender, white fills, hatched accents
